# Code of Conduct

## Our Standards

Be respectful.

Examples of acceptable behavior:

- Constructive feedback
- Respectful communication
- Professional conduct

Unacceptable behavior includes:

- Harassment
- Abuse
- Discrimination
- Trolling

## Enforcement

The maintainer may remove:

- Comments
- Issues
- Pull Requests

or block users who violate this Code of Conduct.

## Scope

This applies to:

- Issues
- Pull Requests
- Discussions
- All project spaces

## Contact

Contact the maintainer via E-Mail (contact@robinkutzner.de) if necessary.
