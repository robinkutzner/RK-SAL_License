---

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

### Preferred method

Use **GitHub Security Advisories**:

- Go to the repository
- Click **Security**
- Click **Report a vulnerability**

This allows private reporting.

---

## What to include

Please include as much information as possible:

- Description of the vulnerability
- Steps to reproduce
- Proof of concept (if available)
- Impact assessment
- Affected version(s) or commit hash

---

## Response Policy

The maintainer will make a reasonable effort to:

- Acknowledge the report within **7 days**
- Investigate and assess the issue
- Provide a fix or mitigation if applicable

Response time is not guaranteed.

---

## Responsible Disclosure

Please do **not publicly disclose** the vulnerability until:

- A fix has been released, or
- The maintainer has confirmed that disclosure is acceptable.

This helps protect users of the Software.
