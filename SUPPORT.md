# Support

## Getting Help

If you need help, please use:

- GitHub Issues (preferred)

## Before opening an issue

Please check:

- README
- Existing Issues

## What to include

Please provide:

- Clear description
- Steps to reproduce
- Expected behavior
- Actual behavior
- Version / commit

## Response Time

Support is provided on a best-effort basis.

Response is not guaranteed.

## Commercial Support

Commercial support may be available under separate agreement.
